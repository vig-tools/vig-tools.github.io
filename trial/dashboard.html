Usage logs are written to CloudWatch Logs under the log group /aws/lambda/vig-trial-auth. To view them:
1. Open the CloudWatch Logs console
2. Select the log stream for the date you want
3. Filter by TOKEN_ACCEPT, USAGE_QUERY, or USAGE_CALC

Each log entry is a JSON object with: event, company, token, detail, ip, timestamp.
